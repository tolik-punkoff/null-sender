НАЗНАЧЕНИЕ ПРОГРАММЫ
Программа предназначена для боевых магических ритуалов.
СИСТЕМНЫЕ ТРЕБОВАНИЯ
Компьютер: PIII 700, 256 Mb ОП и выше
Операционная система: Microsoft Windows XP SP1 (32 и 64-разрядная) и выше
Дополнительные компоненты: Microsoft .NET Framework версии 2.0 и выше

КАК ИСПОЛЬЗОВАТЬ ПРОГРАММУ
0. Подготовься к ритуалу, придумай соответствующие заклинания, 
сосердоточься на тех объектах, которые хочешь уничтожить.
1. Запусти программу. При запуске появится изображение Черного Солнца.
Старайся смотреть в его центр.
2. Внеси необходимые объекты в список, внеся их наименования в текстовое поле,
и нажав кнопку "Добавить"
3. В случае ошибки или опечатки удали ненужные объекты из списка, нажав кнопку 
"Удалить"
4. Произнеся соответствующее ритуалу заклинание нажми на кнопку "Приступить",
предварительно отринув лишние мысли.

КАК РАБОТАЕТ ПРОГРАММА:
С теоретическим обоснованием работы программы можно ознакомиться здесь:
http://lj.rossia.org/users/hex_laden/39379.html
Технически же, программа позволяет тебе указать объекты для уничтожения,
после чего, объединив объекты в строку, она генерирует образ хаос-компоненты
в виде случайного числа, и присоединив его к данной строке, отправляет строку
в нуль-устройство, тем самым, ритуально уничтожая указанные тобой объекты.

УСЛОВИЯ РАСПРОСТРАНЕНИЯ
Программа распространяется бесплатно, с открытым исходным кодом.
Разрешается модифицировать программу в целях ее улучшения.
Запрещается получать материальную выгоду от распространения программы.
Получение материальной выгоды, коммерческое использование программы карается
Богами Хаоса.
